# HSK 构句大师 · باني الجُمل — Web App

Play in the browser (no install): **https://man072012.github.io/hskgame-web/**

A gamified HSK1–4 **sentence-ordering** trainer (LEGO/jigsaw style), Arabic UI.
- تمهيدي (shape-fit jigsaw), ترتيب (tap-to-place), بطاقات الصور (96–100).
- Long-press any piece → grammar role + positional rule + sentence explanation.
- 📖 «كلمات الجملة» → tap a word → full card (Arabic meaning, pinyin, **audio + image**, example, mnemonic). 1201 words (HSK1–4), fully offline.

**Android APK:** [HSK4-Builder.apk](HSK4-Builder.apk) (≈38 MB, signed).

Single file `index.html` + `img/` (picture-writing) + `vocab/aud` + `vocab/img` (word media).
Word audio/images are extracted from the author's HSK Anki decks (HSK标准教程-based).
